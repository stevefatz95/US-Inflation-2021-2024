# Understanding Inflation Dynamics in the United States: An Analysis of 2021–2024

This repository contains the dataset and reference notebook used in:

**Grassi, S. (2024). _Understanding Inflation Dynamics in the United States: An Analysis of 2021–2024_.**  
[ResearchGate Link](https://www.researchgate.net/publication/384766504_Understanding_Inflation_Dynamics_in_the_United_States_An_Analysis_of_2021-2024)

The materials are intended for **research, education, and applied data analysis practice**, focusing on the evolution of U.S. inflation during the post-COVID period.

---

## Contents

- **`inflation-dataset.xlsx`** — Monthly Headline and Core PCE inflation data (2021–2024)  
- **`us_inflation_analysis.ipynb`** — Jupyter Notebook used to generate the analysis and figures presented in the paper  

---

## Dataset Summary

- Monthly Headline and Core PCE inflation rates  
- Coverage: 2021–2024  
- Data is aggregated and contains no sensitive information

---

## Data Source

- Publicly available U.S. inflation data compiled from official sources, including the Federal Reserve Economic Data (FRED)
